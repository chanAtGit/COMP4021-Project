**Game Title: SHOWDOWN**
- Fast-paced 1v1 top-down shooter
- Match time: 3 minutes. 
- Weapons and items spawn randomly on the ground
- Last 30 seconds - Sudden Death - Remove all obstacles
- Last 15 seconds - Sudden Death - One hit one kill
- If players survive sudden death, draw
- Each player has two slots - (Slot 1 - weapon, Slot 2 - item)
